## 管理指南

### 概述

drouting 模块提供动态路由功能，用于根据可配置的目标模式匹配将呼叫路由到不同的目的地。

### 依赖

#### OpenSIPS 模块

以下模块必须在此模块之前加载：

- *tm* -- 事务模块

#### 外部库或应用程序

运行加载此模块的 OpenSIPS 之前必须安装以下库或应用程序：

- *无*

### 导出的参数

#### drouting 模块参数

本模块无导出参数。

### 导出的函数

#### drouting 模块函数

本模块无导出函数。
